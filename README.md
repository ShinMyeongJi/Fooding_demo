# Fooding_Client_Demo(1차 개발 완료)

## 간단소개
관리자가 관리하는 각 음식점의 메뉴들을 주문 배달하는 메커니즘을 따라 만들어본 어플입니다.(실제 결제 모듈 X)

## 개발환경
* Language : Kotlin, Javascript
* DB : Firebase
* Web Server : Apache
* Protocol : HTTP
* Android Studio @3.6.1

## Application 버전
* minSdkVersion 21
* targetSdkVersion 29
